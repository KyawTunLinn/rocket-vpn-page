Installation Guide

Follow these simple steps to install Rocket VPN on your server.

üöÄ Quick Install

Once you have the `setup.sh` script (from the repository), run it as root:

# 1. Install Necessary
apt update && sudo apt upgrade -y
apt install curl wget unzip -y
# 2. Download the installer
wget https://github.com/KyawTunLinn/rocket-vpn/releases/download/rocket-vpn-0.0.2/rocket-vpn-0.0.2-installer.zip
# 3. Unzip the installer
unzip rocket-vpn-0.0.2-installer.zip
# 4. Change directory to the installer
cd rocket-vpn-0.0.2-installer
# 5. Make the installer executable
chmod +x setup.sh
# 6. Run the installer
./setup.sh

This script will automatically:

  • Install WireGuard¬Æ kernel modules and tools
  • Install Java 21 (OpenJDK)
  • Enable IP Forwarding
  • Configure the systemd service for Rocket VPN
  • Start the Rocket VPN service
üìã Requirements
  • Debian 12 and above or Ubuntu 24.04 and above
  • Root privileges
  • Port 51820/UDP (WireGuard)
  • Port 8080/TCP (Web UI)
